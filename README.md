# ABANDONED

NewConsole
==========

What's this?
------------

Simple console replacement for Windows.

Note
----

You have to run this program in native bitness (x64 build on x64 windows, ..)

Features
--------

- UTF-8 console 
- CJK input support (Using TSF)
- Alpha background support
- Many bugs
